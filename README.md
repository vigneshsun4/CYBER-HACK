# CYBER HACK

CYBER HACK is a small 2D platformer where you move through a corrupted digital system trying to reach the core of the network. The game mixes classic platforming with simple hacking mechanics and movement tools like grappling.

The idea is that you're navigating through a broken network environment controlled by a rogue AI. Along the way you’ll run into system defenses, locked terminals, and sections that require careful movement to get past.

---

# Gameplay

The game is a side-scrolling platformer. You move through the level by jumping between platforms, avoiding hazards, and interacting with parts of the system.

While progressing you’ll find things like:

- terminals that can be hacked to change the environment
- enemies that patrol parts of the system
- collectibles representing recovered data
- sections where grappling is required to move forward

The goal is simply to make it through the system and reach the final section of the network.

---

# Mechanics

## Movement

Movement is based on classic platformers.

You can:

- move left and right
- jump between platforms
- wall slide and wall jump in certain areas

---

## Grappling

Some blocks act as grapple anchors.

Press **G** near them to fire a grapple and pull yourself toward that point.  
This helps cross large gaps or reach higher platforms.

---

## Hacking

Certain terminals can be interacted with.

Press **H** near a terminal to start a small hacking challenge.  
When successful, it can trigger things like moving platforms or opening paths.

---

## Collectibles

Throughout the level there are data chips scattered around.

Collecting them increases your chip count and represents recovering pieces of the system that were corrupted.

---

## Enemies

There are a couple of simple enemies in the game.

### Patrol Bots
These move back and forth across platforms and damage the player on contact.

### Firewall Turrets
These stay in place and periodically fire projectiles.

---

# Controls

| Key | Action |
|-----|------|
| A / D or ← → | Move |
| Space | Jump |
| G | Grapple |
| H | Hack terminal |

# Running the Game

Clone the repository:

```bash
git clone https://github.com/yourusername/cyber-hack
```

Then open `index.html` in your browser.

The game runs entirely in the browser using HTML5 canvas, so nothing needs to be installed.
